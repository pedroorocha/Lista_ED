# 🔧 Executando o projeto com ambiente virtual (venv)
1️⃣ Criar o ambiente virtual dentro da pasta do projeto

python -m venv .venv

2️⃣Se necessario Ativar o ambiente virtual

Sistema Operacional	Comando
Windows (PowerShell)	.\.venv\Scripts\Activate.ps1
Windows (CMD)	.\.venv\Scripts\activate.bat
Linux / Mac	source .venv/bin/activate

3️⃣ Rodar o projeto

Ex: python Questão_1.py
